# FitMeet Social Orbit 首页分镜 V5.0

首页分镜保持现有获奖级方向。全站扩展后，固定导航中的 `WORLD / MOMENTS / AGENT / SAFETY / APP` 进入真实二级页面；首页内部章节通过场景 CTA 和滚动继续访问。跨页转场遵循 `FITMEET_SITE_ARCHITECTURE.md`。

## 总体节奏

```text
Hero / Image Constellation       0–220svh
World Reel / 4 Moments           overlap 120svh + 520svh
Social Signal                    overlap 100svh + 150svh
Agent in the World               overlap 100svh + 116svh
Trust                            112svh
Final Download                   100svh
```

## 1. Hero — 影像星图

- 暖白背景、固定胶囊导航
- 中心：`SOCIAL WORLD / 让社交更简单`
- 10 个影像碎片：旅行、城市运动、露营、球场、公共空间、摄影
- 低对比 SVG 路径穿过碎片，光谱脉冲沿轨迹移动
- 底部 `APP · COMING SOON` 初始为收起状态

滚动后：碎片向空间外退去，夜间运动主图从左下角持续放大；主图全屏后才出现章节文案与双商店舱。

## 2. World Reel — 四个全屏时刻

1. `01 / AFTER WORK`：夜晚不散场，城市仍在回应。
2. `02 / WEEKEND`：同路的人，让目的地更近。
3. `03 / OUTDOOR`：一起抵达，远方才成为记忆。
4. `04 / CITY`：兴趣相同的人，会让一座城变熟。

四幕位于同一个固定画布。下一幕通过椭圆曲线蒙版覆盖上一幕；图片先进入，主标题后逐字出现。底部四段进度线只强调当前章节。

## 3. Social Signal

- 延续 City 的同一张公共空间摄影
- 六张碎片重新出现并从分散位置回到轨道
- SVG 路径依次连接时间、地点和人物节点
- 文案：`你说一句，世界开始组织。`
- 需求：`周末想找旅行搭子。`
- 状态：时间已理解 / 地点已连接 / 同路的人正在出现

章节结束时整个 Signal Stage 淡出，背景下方已经是 Agent 世界，不出现硬切。

## 4. Agent in the World

- 全屏真实品牌视频，暗色处理
- 左侧：品牌主张与产品边界
- 右侧：单一 38vw 交互控制台
- 支持需求选择、生成计划和确认计划
- 视频进入视口后播放，离开后暂停

## 5. Trust

- 全屏城市运动摄影
- 标题：`真实发生，也有清晰边界。`
- 底部开放式三列：需求由你定义 / 关系保持真实 / 安全先于效率
- 不使用卡片背景

## 6. Final

- 近黑色连续空间
- Social Orbit 路径汇聚至品牌 Logo
- 标题：`让下一次出发，不再等一个碰巧。`
- App Store / Google Play 完整上线舱
- 页底保留版权、返回顶部和品牌说明
